continue 文を使おう

## for ループの途中をスキップさせる

プログラム実行結果になるように、次のプログラムの for 文中に continue 文をコーディングしなさい

``` java
package jp.freestyles.day.sixteen;

public class ForLoopBreaker {
    public static void main(String[] args) {

        for (int i = 1; i <= 10; ++i) {
            
            System.out.println(i);
        }
    }
}
```

### 出力結果

```
1
2
3
4
9
10
```

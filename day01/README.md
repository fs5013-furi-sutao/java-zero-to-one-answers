# 1 日目

変数を理解しよう

## Practice 01: 名前と年齢を表示する

プログラム実行結果例になるように、次の処理をコーディングしなさい

- 変数 myName に「山本一郎」を代入
- 変数 age に 757 を代入
- myName と age を表示

### 出力結果

```
私は 山本一郎 です
年齢は 757 歳です
```

## Practice 02: 3 つの文字列を表示する

プログラム実行結果例になるように、次の処理をコーディングしなさい

- 変数 firstBlock に「AAA」を代入
- 変数 secondBlock に「bbbb」を代入
- 変数 thirdBlock に「CCCCC」を代入
- firstBlock を表示
- secondBlock を表示
- thirdBlock を表示

### 出力結果

```
AAAbbbbCCCCC
```
